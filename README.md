# Sathyajith — EEE Portfolio

Personal portfolio of Sathyajith, B.E. Electrical & Electronics Engineering  
PSG College of Technology, Coimbatore (2023–2027)

Built with React + Vite. Fault Diagnostic Terminal design theme.

## Stack
- React 19 + Vite 8
- Plain CSS-in-JS
- Fonts: Orbitron, Share Tech Mono

## Run locally
npm install
npm run dev
